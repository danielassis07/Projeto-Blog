C++ ultrapassa Java em ranking de programação; veja lista

Postado em 06 de Junho 2023 - 18h25

Todo ano, o índice TIOBE da Comunidade de Programação decide quem leva o prêmio de linguagem do ano. Em 2022, a principal novidade é o C++ figurando entre as mais populares, junto ao Python e o C. É a primeira vez em que ele ultrapassa o Java na história do índice, que começou em 2001. Python e C já são figurinhas carimbadas no prêmio, tendo levado diversas vezes. O C++ só alcançou o feito há quase 20 anos, em 2003. Com a ascensão desta linguagem, é a primeira vez em que Java não está no pódio. Apesar disso, esta última também cresceu em 2022.“O desempenho ainda é muito importante hoje em dia e C++ é a linguagem de programação de sistemas grandes mais rápida que existe”, explicou Paul Jansen, CEO do Tiobe. A linguagem surgiu em 1979, criada por Bjarne Stroustrup, e foi padronizada em 1998.
